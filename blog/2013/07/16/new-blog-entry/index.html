New Blog Entry

Jul 16th, 2013

This is my first attempt to create a blog post. The first thing I just learned was how to rake a new post. After typing rake new_post[“New Blog Entry”] I realized it created the file within RubyMine.

Next I need to learn how to push the code to Github. After reading through the presentation I learned about rake deploy. Let’s see if this works?

-Update-

Followed these instructions:

Deploying to Github Pages • Create a new repo on Github titled “username.github.com” • rake setup_github_pages – Enter the SSH git URL from the previous step • rake generate • rake deploy

Everything worked. Now, how do I update the deployed site?
